Angular Custom Elements

This component was created using Component Driven Development paradigm with one other software developer. We used Storybook to build and test our work in a component driven fashion.

  • It uses CDK Virtual Scroll
  • It lazy loads an infinite list of news articles from the NewsAPI.org
  • It is complied to pure javascript and can be included in any web enable project -- Even one that isn't using angular.

    • Custom Elements are a great solution for large organizations that have made a big investment in legacy code and can't change their tooling every six months but still would like to take advantage of the most advanced tools and methodologies.

    View the source of this page to see the custom element tag. Behind the scenes I'm lazy loading the javascript bundle. One thing to keep in mind is that angular custom elements can be a little bit large (this one is).

    If you select a new newsource or bring up the settings dialog you will see there is one issue with the Overlay Component. This is an open bug we are discussing. If this element is not working for any reason it is because too many people have loaded news articles and I'm over my limit with the good people at NewsAPI.org.
